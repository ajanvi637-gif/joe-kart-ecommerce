# 🛒 Joe Kart - Ecommerce Web Application

## 📌 About the Project
Joe Kart is a modern ecommerce web application built using the MERN Stack (MongoDB, Express, React, Node.js).  
It provides a smooth shopping experience with user-friendly UI and responsive design.

---

## 🚀 Features
- User authentication (Login/Signup)
- Product listing with images
- Add to cart functionality
- Order management
- Responsive UI for all devices
- Fast and dynamic performance

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript, React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Version Control: Git & GitHub
